课程简介

鉴于 Access mdb应用程序开发 SQL Server 数据库应用程序的弱点，Microsoft 公司在 Access 2000 中增加了一种新的Access 应用程序格式——ADP，该格式称为Access 数据库项目，以提高 Access 应用程序的开发效率。
这种应用程序只能针对 Microsoft SQL Server 数据库进行开发，放弃 Jet-Engine 和 DAO数据访问组件，采用 ADO 数据访问组件，将连接的数据库对象映射到ADP 中，因此ADP 可以直接对SQL Server 数据库进行操作。ADP 本身不存储任何数据。
应用程序的开发和操作界面与 mdb应用程序非常相似，大大降低了办公人员对 SQL Server 数据库数据处理的难度，大大提高应用程序的开发效率。#### 课程目标：1、学会使用ADP来开发出网络版的应用程序2、学会Access与SQL Server交互使用，使系统达到更高的标准。打破传统的观念，让Access更加满足我们的开发需求。适合对象:1、学习Access一段时间的工作人员，对Access有一定的了解及想提高水平的2、需要开发一套能共享数据的Access应用系统的从业人员3、刚学习access的新手或者学生，对Access抱有浓厚的兴趣，且不怕困难，努力学习的学习条件：有无基础均可学习使用（最好能懂基本的操作）
