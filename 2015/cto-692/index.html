课程简介

在本系列课程中，涉及到微软全新一代的产品。包括Windows Server 2012，Exchange Server 2013，Lync Server 2013，SharePoint Server 2013，SQL Server 2012，SCCM 2012的产品，本次系列课程中会针对这些产品通过案例来做全面的讲解。 通过本次系列课程能够了解微软新一代产品有哪些新特性、这些新特性应用的时候有哪些注意点，以及这些新特性能帮助企业带来什么，在本次课程中会看到答案。 本次系列课程除了有联科讲师 Steven.Wang，Alex.Yang，Klad.Xue外，还有万众瞩目的微软原厂技术顾问鱼跑跑给大家介绍全新的微软、全新的微软产品和微软发展的方向和趋势。
