课程目标

绝对不要人肉部署！ 务必一切自动化！ Capistrano就是这一领域最好的工具！

适用人群

开发人员，运维人员，适合所有老兵 + 新手。

课程简介

在程序员的世界， 我们要追求极致， 效率，准确。时间就是生命！可惜的人是，人肉根本不行. 效率低，　速度慢，　特别容易出错．你的Java程序，是不是在Eclipse中打包成war, 然后　ftp上传到远程？然后还需要手动启动tomcat? 你的PHP程序，是不是也是写好之后，ftp上传到远程？ 或者 使用 $svn co, 或者 $git pull? 你的H5程序, 是不是直接上传到远程服务器的某个目录? 如果用 部署的水平,来衡量程序员, 可以认为大概分成5种: 1. 最笨的程序员是不会做部署的. 没有摸过服务器, 所有的代码都是在本地运行的 “开发模式”. 2. 新手程序员, 都是通过FTP这样的工具来上传代码.3. 入行一段时间的程序员, 知道了 “版本控制”这个工具的, 就会在远程使用 $ git pull 或者 $ svn checkout 这个命令, 来获取最新的代码.4. 再熟练一些的程序员 ,以及运维, 慢慢体会到了”上传ftp, 部署时间太久,麻烦”这样的痛点, 开始写自动化的脚本了.5. 程序员老鸟, 经历过 “生产事故”, 需要”30秒内回滚代码” 这样的痛点的人, 才知道, 部署不但要自动化, 回滚也要非常方便才行. 我们需要一种工具, 可以让我们: 1. 能够自动化部署. 一行命令, 就可以实现2. 简单. 3. 能为一切 web语言做自动化部署, 例如: Java, PHP, HTML, Vuejs, Angular, Python, Ruby 等等. 4. 出了问题可以方便的回滚.Capistrano (原意是一座美丽的城市). 就是我们想要的工具
